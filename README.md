## 📦 Installation

Clone the repo and get started:

```sh
# Clone the repository
git clone https://github.com/yourusername/calculator.git
cd calculator
```

### For Python
```sh
# Run the calculator
python calculator.py
```

### For Web (React-based)
```sh
npm install
npm start
```
